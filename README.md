# Kernel_svm_tutorial

This repository contains my machine learning tutorial assignment on Support Vector Machines (SVM), with a focus on comparing linear, polynomial, and RBF kernels.

## Files
- kernel_svm_tutorial.ipynb – Jupyter notebook containing the code, model training, evaluation, and plots
- seeds_dataset.txt – dataset used in the notebook
- SVM_Kernal_Report.pdf – tutorial report submitted for the assignment

## How to run
1. Download the repository files.
2. Keep kernel_svm_tutorial.ipynb and seeds_dataset.txt in the same folder.
3. Open the notebook in Jupyter Notebook or JupyterLab.
4. Run all cells from top to bottom.

## Topic
This tutorial explains how different SVM kernels affect classification performance and decision boundaries, using the Seeds dataset as the main experiment and a synthetic moons dataset as a teaching example.
